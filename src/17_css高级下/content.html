两列布局

				  1.  两列自适应宽度 ;

				  2.  由于div为块状元素，默认情况下占据一行的空间，要想让下面的div跑到右侧就需要做助css的浮动来实现 ;

				  3.  两列固定宽度 ;

				  4.  两列固定宽度居中 （在两个div之外再加一个父div）;
			
					<div style="width:100px; height:70px; background:#9f9; float:left; "></div>
					
					<div style="width:100px; height:70px; background:#9ff; float:left; "></div>
				三列布局

				  1.  三列自适应宽度 （左列和右列固定，中间列根据浏览器宽度自适应）;

				  2.  三列固定宽度 ;

				  3.  三列固定宽度居中 ;
			
					<div style="width:50px; height:70px; background:#9f9; float:left; "></div>
					
					<div style="width:100px; height:70px; background:#9ff; float:left; "></div>
					
					<div style="width:50px; height:70px; background:#9f9; float:left; "></div>
