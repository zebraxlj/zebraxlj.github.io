1 声明DTD类型标签：!DOCTYPE   from: w3schools.com

a. The <!DOCTYPE> declaration must be the very first thing in your HTML document, before the <html> tag.
b. The <!DOCTYLE> declaration is not an HTML tag; it is an instruction to the web browser about what version of HTML the page is written in.
Tip: The <!DOCTYPE> declaration is NOT case sensitive.

2 html 标签   from: w3school.com

a. The <html> tag tells the browser that this is an HTML document.
b. The <html> tag represents the root of an HTML document.
c. The <html> tag is the container for all other HTML elements (except for the tag).

3 head 标签   from: w3school.com

a. The <head> element is a container for all the head elements.
b. The <head> element can include a title for the document, scripts, styles, meta information, and more.
c. The following elements can go inside the <head> element:

  • <title> (this element is required in an HTML document)
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>

4 meta 标签(charset属性值，Keywords属性，description属性)

a. Metadata is data (information) about data.
b. The <meta> tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable.
c. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.
d. The metadata can be used by browsers (how to display content or reload page), search engines (keywords), or other web services.

图像img标签   from:

a. The <img> tag defines an image in an HTML page.
b. The <img> tag has two required attributes: src and alt.
Note: Images are not technically inserted into an HTML page, images are linked to HTML pages. The <img> tag creates a holding space for the referenced image.
Tip: To link an image to another document, simply nest the <img> tag inside <a> tags.

ul无序列表   from: w3school

a. The <ul> tag defines an unordered (bulleted) list.
b. Use the <ul> tag together with the <li> tag to create unordered lists.
Tip: Use CSS to style lists.
Tip: To create ordered lists, use' the <ol> tag.

ol有序列表   from: w3schools.com

a. The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.
b. Use the <li> tag to define list items.
