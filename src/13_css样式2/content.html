CSS 文本属性定义文本的外观
				
				  <1> 文本的颜色（color ，背景） ;

				  <2> 字符间距 （letter-spacing: 20px; letter-spacing: -0.5em;）;

				  <3> 行间距 （line-height: 30px; line-height: 200%; line-height: 2;）;

				  <4> 对齐文本 （text-align: center/left/right） ;

				  注：16px=1em, 将原来的px数值除以16，然后换上em作为单位就ok
				      em 是相对尺寸，1（em） = 当前位置文字大小（px）

				  <5> 装饰文本 text-decoration; 
					

						

							值
							描述
						

						

							none
							默认，定义标准的文本。
						

						

							underline
							定义文本下的一条线
						

						

							overline
							定义文本上的一条线
						

						

							line-through
							定义穿过文本的一条线
						

						

							blink
							定义闪烁的文本（deprecated）
						

						

							inherit
							规定应该从父元素继承 text-decoration 属性的值
						

					

				  <6> 对文本进行缩进 text-indent（将段落的第一行缩进）;
					

						

							值
							描述
						

						

							length
							定义固定的缩进，默认：0
								
text-indent:30px;

									
						

						

							%
							定义基于父元素宽度的百分比的缩进
								
text-indent:10%;

									
						

					

			
				控制文本中的字母

				  text-transform 属性控制文本的大小写
				  capitalize，uppercase，lowercase
					

						

							值
							描述
						

						

							none
							默认。定义带有小写字母和大写字母的标准文本。
						

						

							capitalize
							额我那本中的每个单词以大写字母开头。
						

						

							uppercase
							定义仅有大写字母。
						

						

							lowercase
							定义无大写字母，仅有小写字母。
						

					

			
				CSS 字体属性定义文本

				  <1> 文本的大小（font-size）

				  <2> 字体系列 （font-family）

				  <3> 字体风格（font-style：normal / italic / oblique）
					

						

							值
							描述
						

						

							normal
							默认值，浏览器显示一个标准的字体样式
						

						

							italic
							浏览器会显示一个斜体的字体样式
						

						

							oblique
							浏览器会显示一个倾斜的字体样式

				CSS 表格样式：

				  <1> 表格边框 (border) ;
				      Ex: border:1px solid red;
				  <2> 折叠边框 (border-collapse:collapse;);

				  <3> 表格宽度和高度 (width, height);

				  <4> 表格文本对齐 (text-align);
				      Ex: text-align:center;
				  <5> 表格内边距 (padding);

				  <6> 表格颜色 (background);
